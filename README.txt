Employee Management Application
Introduction
The Employee Management Application is a Java-based software designed to simplify employee information management. It empowers HR departments and managers to add, view, update, and delete employee records efficiently.

Features
Add Employee: Add a new employee with details.
View Employee: Retrieve employee details by ID.
Update Employee: Modify an employee's information.
Delete Employee: Remove an employee from the database.
View All Employees: Display a list of all employees.
Exit: Close the application.
Requirements
Java Development Kit (JDK) 8 or higher
A terminal or command prompt
Files
Employee.java: Defines the Employee class.
EmployeeService.java: Contains methods for managing employee data.
Main.java: Contains the main method and menu logic to interact with the user.
